# Board Game Helper (BGH)

A mobile-first board game score tracker, built with React + TypeScript + Tailwind CSS.

**Live:** https://sword-jin.github.io/bgh/

## Supported Games

### Flip 7
- 2–12 players with distinct colors and editable names
- Round-by-round score input via bottom sheet
- Live rankings with per-round breakdown
- Edit past rounds by tapping round chips
- Game ends when any player reaches 200+ points — winner and runner-up announced

## Development

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` auto-deploy to GitHub Pages via GitHub Actions.
